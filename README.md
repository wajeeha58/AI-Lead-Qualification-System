# AI-Lead-Qualification-System

## AI Lead Qualification System Using n8n, Gemini AI, and Airtable

This project automates the lead qualification process using AI. Users can submit lead information in natural language, and the workflow automatically extracts lead details, assigns a lead score, categorizes the lead (Hot, Warm, or Cold), recommends the next action, and stores the results in Airtable CRM.

## Key Features

- Lead information extraction from natural language
- AI-powered lead scoring
- Hot/Warm/Cold lead categorization
- Recommended next actions
- Structured JSON output parsing
- Automatic Airtable CRM updates
- End-to-end workflow automation using n8n

## Tech Stack

- n8n
- Google Gemini AI
- Airtable

## Workflow

User Input → Gemini AI → Structured Output Parser → Airtable CRM
